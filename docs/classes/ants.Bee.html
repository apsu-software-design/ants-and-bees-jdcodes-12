  • Overrides Insect.act

    Performs a Bee's sting action during the turn. Checks to see if an Ant is blocking it's path. If an Ant is present, and the Bee is not under the cold special condition it calculates damage to the Ant that is blocking it.

    Additionally checks to see if the Bee needs to retreat (i.g. removed from tunnel)
